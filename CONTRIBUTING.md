# Contributing to WCP

WCP is an open standard. Contributions to the specification are welcome.

## How to contribute

1. Open an issue at https://github.com/fafolab/wcp/issues
2. Discuss the change
3. Submit a pull request with edits to WCP_SPEC.md

## Reference implementation

[PyHall](https://pyhall.dev) (Python) is the reference implementation.
Other implementations: [@pyhall/core](https://www.npmjs.com/package/@pyhall/core) (TypeScript), [pyhall-go](https://github.com/fafolab/pyhall-go) (Go)
